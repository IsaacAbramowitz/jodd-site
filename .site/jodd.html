Jodd's Periodic Table of Components

Here are some components for you! See documentation for more.
B
BeanUtil - fast bean library, type introspection and type conversion tools with unique features.
T
JDateTime - elegant usage and astronomical precision in single time-manipulation class.
R
Props - super properties replacement, featuring: UTF8, sections, profiles, macros and more.
D
DbOom - efficient and thin DB layers and object mapper that simplify database code.
M
Madvoc - elegant web MVC framework based on CoC and annotations for pragmatic web development.
P
Petite - slick and lightweight DI container supports sufficient most of features offered by other containers.
X
Proxetta - the fastest proxy creator with new, unique approach for making pointcuts and different type of advices.
L
Lagarto - super fast and versatile all-purpose HTML parser and DOM builder.


J
Jerry - jQuery-friendly HTML parser with CSS3 selectors support.


H
HTTP - Tiny and raw HTTP client, helps talking to servers.

E
Email - sending and receiving emails for busy developers.

C
Decora - powerful decoration framework for web pages based on templates.
